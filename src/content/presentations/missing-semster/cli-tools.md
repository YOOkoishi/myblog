# 常用命令行工具

## 为什么需要工具?

> "工欲善其事，必先利其器"

- 提高效率
- 改善体验
- 解决特定问题

---

## 文件检索

### fzf (Fuzzy Finder)
- 模糊搜索文件、历史命令
- 交互式界面
- `Ctrl+r` 增强

### ripgrep (rg)
- 极速文本搜索
- 自动忽略 .gitignore
- 比 grep 快得多

---

## 系统监控

### htop / btop
- 可视化的进程管理
- CPU、内存、网络实时监控
- 支持鼠标操作

### ncdu
- 磁盘占用分析
- 交互式清理文件

---

## 开发辅助

### tldr (Too Long; Didn't Read)
- 简化的 man 手册
- 只展示常用示例
- `tldr tar`

### bat
- 带语法高亮的 cat
- 集成 Git 状态
- 自动分页

---

## JSON 处理

### jq
- 命令行 JSON 处理器
- 格式化输出
- 提取字段
- `curl ... | jq .`

---

## 总结

- 多尝试新工具
- 建立自己的工具箱
- 编写脚本自动化工作
