# 第一章：介绍

## 什么是演示文稿

演示文稿是一种用于展示信息的工具，通常用于会议、教学、分享等场景。

- 可以包含文字
- 可以包含图片
- 可以包含代码

## 为什么使用 Reveal.js

Reveal.js 是一个强大的演示框架：

1. 支持 Markdown 语法
2. 支持代码高亮
3. 支持数学公式
4. 响应式设计

## JavaScript 代码

这是一段简单的 JavaScript 代码：

```javascript
function hello(name) {
  console.log(`Hello, ${name}!`);
}

hello("World");
```

## Python 代码

这是一段 Python 代码：

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```
