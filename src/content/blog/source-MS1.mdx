## Part II . 为什么要学习Linux

**问题来了：**

> 我每天都用 window。

> OpenEuler 好难装

> 我为什么要学习 Linux？


<PresentationViewer 
  src="/presentations/missing-semster/whylearinglinux.html" 
  title="为什么要学习Linux"
  description="why learning linux"
  height="500px"
/>


此外推荐一些书籍和文档
📚：

- **《Linux 命令行与 Shell 脚本编程大全》（第4版）**
  - 作者：Richard Blum, Christine Bresnahan
  - 适合入门和进阶，涵盖命令行基础、shell脚本、自动化等内容。

- **《鸟哥的 Linux 私房菜：基础学习篇》**
  - 作者：鸟哥（吴羽）
  - 中文经典，系统性强，适合零基础用户。

- **《Linux 就该这么学》**
  - 作者：刘遄
  - 讲解通俗易懂，配套实验丰富，适合自学。

- **《UNIX/Linux 系统管理技术手册》**
  - 作者：Evi Nemeth 等
  - 适合进阶和运维方向，内容全面。

- **《Linux Pocket Guide》**
  - 作者：Daniel J. Barrett
  - 便携式参考手册，查找命令很方便。

---

🌐 在线文档

- [Linux Command](https://linuxcommand.org/)
  - 英文，系统介绍命令行和 shell 脚本基础。

- [The Art of Command Line](https://github.com/jlevy/the-art-of-command-line)
  - GitHub 上的命令行艺术，涵盖实用技巧和进阶用法。

- [鸟哥的 Linux 私房菜在线版](https://linux.vbird.org/)
  - 免费中文教程，内容详实。

- [Linux Manual Pages](https://man7.org/linux/man-pages/)
  - 官方英文 man 手册，查找命令参数和用法。

- [Linux 就该这么学官网](https://www.linuxprobe.com/)
  - 配套实验和视频教程。

- [Linux Journey](https://linuxjourney.com/)
  - 英文互动式学习网站，适合入门。

- [菜鸟教程 Linux 教程](https://www.runoob.com/linux/linux-tutorial.html)
  - 中文基础教程，适合快速查阅。
